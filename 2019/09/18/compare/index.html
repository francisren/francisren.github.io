python比较符号

运算符 含义
< 判断左边是否小于右边
<= 判断左边是否小于等于右边
> 判断左边是否大于右边
>= 判断左边是否大于或等于右边
== 判断左右两边是否相等
!= 判断左右两边是否不相等
is 判断两个对象的id是否相等
is not 判断两个对象的id是否不相等
